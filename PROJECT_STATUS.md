## 🚀 **How to Start**

### Option 1: Quick Start (Windows)
```batch
scripts\start_web.bat
```

### Option 2: Cross-Platform
```bash
python scripts/start_web.py
```

### Option 3: Manual Start
```bash
# Terminal 1 - Backend
cd backend
python main.py

# Terminal 2 - Frontend  
cd frontend
python -m http.server 8080
```

Then visit: **http://localhost:8080**
